We know ,JavaScript can get and change any object property .So that , JavaScript covert HTML and CSS file in a Document Object Model and that way, JavaScript can achive full access of HTML and CSS and can change the document structure, style, and content.

What is a callback function and why will you use it?

Ans:A callback function in javascript is a function that can pass another function as a parameter and calls it inside.It will be executed after the completion of the first function.
